# kotline
Kotline Tutorials

This Kotline Tutorials is been created by me (Rakesh) as an absolute beginner in the field of Android Development.

So anyone who is interested in learning Android App Development can follow along.

To learn Kotline :
1. Go to rketech/kotline repository
2. Then select master from the dropdown where you see main and a dropdown arrow
3. Click on src
4. There you will get all the .kt files
5. Click on the .kt file(s) as a result you will be able to view the codes
